SECP1513 :
TECHNOLOGY INFORMATION SYSTEM

This course teaches the basics of discrete structure and how it is used in computer science. Students learn about topics like set theory, proofs, relations, functions, recurrence relations, counting methods, graph theory, trees, and finite automata. By the end of the course, students should know how to use things like sets, relations, and functions to solve problems in computer science. They will also learn how to solve problems with recurrence relations and counting, use graph theory and trees to solve real-world problems, and use finite state machines to model electronic devices and other challenges.
